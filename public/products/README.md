# Fotos dos produtos

Coloque aqui as fotos dos equipamentos. O sistema detecta automaticamente.

## Como funciona

Basta salvar a foto com o **mesmo nome do slug do produto** e qualquer uma destas extensões: `jpg`, `jpeg`, `png`, `webp`, `avif`.

### Exemplos

| Produto | Nome do arquivo |
|---|---|
| Fluke 289 Multímetro | `fluke-289-multimetro.jpg` |
| FLIR E8 Termográfica | `flir-e8-termografica.jpg` |
| Keysight DSOX1204G | `keysight-dsox1204g-osciloscopio.jpg` |
| Megger MIT525 | `megger-mit525-isolamento.jpg` |
| Hioki 3169-20 | `hioki-3169-analisador-energia.jpg` |
| Testo 549 | `testo-549-manifold.jpg` |
| Rohde & Schwarz FSH4 | `rohde-fsh4-analisador-espectro.jpg` |
| Fluke 754 | `fluke-754-calibrador.jpg` |

## Recomendações

- **Formato:** JPG ou WebP (menor tamanho)
- **Dimensões:** pelo menos 800×800 px (quadrada funciona melhor)
- **Fundo:** branco ou neutro (fica mais profissional)
- **Peso:** até 300 KB por imagem (o Next.js otimiza automaticamente)

## Se a foto não for encontrada

Aparece uma ilustração esquemática do equipamento (não fica feio, mas o ideal é a foto real).
